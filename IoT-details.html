Automation and Scene Management aspect of Smart Home System

This project was developed as part of a university assignment.

This project development and implementation of an "Automation and Scene Management" system as an integral aspect of a Smart Home System as the Internet of Things (IOT) project. The project's primary objective is to enhance user experience and energy and security efficiency by leveraging advanced reasoning services. The proposed system incorporates dynamic adjustments to lighting conditions based on time of day, weather monitoring, occupancy detection, and a specialized Vacation Mode. The system utilises a combination of proximity sensors and lux sensors integrated with a Raspberry Pi control unit. Through an Android application acting as a display for sensor outputs and providing light control options, users can interact with the system. The central Raspberry Pi hub coordinates sensor inputs and device actions, ensuring seamless integration.
